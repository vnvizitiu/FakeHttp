FakeHttp API

FakeHttp has two namespaces:

1) FakeHttp contains FakeHttpMessageHandler and related types. 2) FakeHttp.Resources contains specific implementations of IReadOnlyResources and IResources for accessing faked responses stored on the file system, in ZipArchives or embedded in an Assembly.
